Solace is hosting a competition in uOttaHack 2020. Simply integrate Solace Cloud into your hack to participate.

Solace Cloud is a software as a service website which hosts the PubSub+ Enterprise Broker which is a message broker used for application level messaging. It’s perfect for any system which requires communication between processes done in a decoupled, reliable fashion.
